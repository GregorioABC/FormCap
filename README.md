# FormCap

O FormCap é um projeto desenvolvido como parte da disciplina de Interação Humano-Computador (IHC). O objetivo do trabalho é aplicar princípios de usabilidade, acessibilidade e design centrado no usuário para criar uma plataforma web educativa e funcional para demonstração de formulários e fluxo de atendimento.

## Objetivo

- Aplicar técnicas e recomendações de IHC para melhorar a experiência do usuário.
- Demonstrar padrões de acessibilidade e usabilidade em interfaces web.
- Prototipar páginas de formulários e telas comuns em processos de atendimento.

## Páginas incluídas

- `index.html` — Página inicial
- `cadastro.html` — Formulário de cadastro
- `usuario.html` — Informações do usuário
- `perfil.html` — Perfil do usuário
- `projeto.html` — Base do projeto
- `responder.html` — Respostan da pesquisa
- `img/` — Imagens e recursos estáticos

> Observação: este projeto é um site estático — para visualizar basta abrir o arquivo `index.html` em um navegador.

## Tecnologias

- HTML5
- CSS3
- JavaScript (quando aplicável)

## Como executar (rápido)

1. Clone ou faça download do repositório.
2. Abra `index.html` no seu navegador favorito.

Se desejar servir localmente por um servidor simples (recomendado para testes de rotas ou fetch):

PowerShell / CMD (se tiver Python instalado):

```powershell
python -m http.server 8000
```

Em seguida abra http://localhost:8000 no navegador.

## Acessibilidade e usabilidade

Este projeto procura seguir boas práticas básicas de acessibilidade: estrutura semântica de HTML, labels em formulários, contraste legível e navegação por teclado sempre que possível. Como exercício de IHC, priorizamos clareza nas interações e feedbacks ao usuário.
